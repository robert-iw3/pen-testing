Overview
Dispatch provides the following 4 user roles administrators can assign when creating new accounts:
  1. Disabled - When a user account is marked as disabled, they will no longer be able to access the web interface and any assigned API keys will not work.
  2. Download Only - Users will only be authorized to download files via web or API. Download Only users have login capabilities but are limited to the "Change-password" page. The index page will remain blank as they are unable to list or edit files, and no other menu options will be provided.
  3. Upload Only - As the name suggests, Upload Only users will only be able to add files using the Web or API. They are not able to list files after upload, modify file permissions, or download files.
  4. Operator - Operators can modify settings, access controls, and add/edit/delete all lower ranking users. However, they cannot interact or view administrators or alternate operator accounts.
  5. Administrator - Super admin account with complete access to the Dispatch application and all users.
Changing Roles

When logged in as an administrator, or operator, navigate to /users and find the chosen user to edit. Select the Role column and choose from the options provided. The user role will automatically update after selecting and flash green to indicate the change was successful.

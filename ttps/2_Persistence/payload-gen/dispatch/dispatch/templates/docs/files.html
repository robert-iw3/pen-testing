Overview
Files must have one of the following 3 permissions applied:
  1. Public - File is accessible to the open internet via alias name.
  2. Public Once - File is accessible one time without authentication, then reverts to Private.
  3. Private - File requires proper authentication via Web UI or API Key.

* If no permissions are set during upload, files will default to Private
* ALL permission types are protected by access restrictions

Changing File Permissions

As an administrator, navigate to the home page (/) containing a list of all files. Find the target file and click the dropdown menu under the Access column. The file will automatically update after selecting the new permission and flash green to indicate the change was successful.
